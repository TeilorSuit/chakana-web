# 🌐 Chakana — Landing Page

Página web oficial del videojuego **CHAKANA**, desarrollada para la **Feria de Saberes Ancestrales** de la UTEQ. Diseñada para ser escaneada vía QR desde dispositivos móviles en el stand de la feria.

> 🎮 Repositorio del juego: [TeilorSuit/chakana](https://github.com/TeilorSuit/chakana)

## Sobre el proyecto

CHAKANA es un videojuego de plataformas 2D basado en la cosmovisión andina. El jugador guía a **Sami** a través de tres mundos — Uku Pacha, Kay Pacha y Hanan Pacha — para reunir los fragmentos de la Chakana y restaurar el equilibrio del universo.

Esta landing page presenta:
- Los personajes principales del juego
- Los desafíos y puzzles de cada mundo
- Las decisiones técnicas y de accesibilidad del proyecto

---

## Stack

- HTML5 puro — sin frameworks, sin build steps
- [Tailwind CSS](https://tailwindcss.com/) vía CDN
- [GSAP](https://gsap.com/) + ScrollTrigger para animaciones
- [Phosphor Icons](https://phosphoricons.com/) vía CDN
- Deploy: GitHub Pages
